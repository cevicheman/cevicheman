## Proyectos

### Escape Estelar

Proyecto desarrollado para la competencia PyWeekend. Se trata de un videojuego de consola donde el jugador debe explorar una estación espacial dañada por una lluvia de meteoritos, recolectar herramientas y recursos, y encontrar la forma de escapar antes de la destrucción total de la nave.

Fue uno de mis primeros proyectos completos y representó una experiencia importante para aprender a estructurar programas más grandes y trabajar con múltiples componentes dentro de una misma aplicación.

**Tecnologías utilizadas**

* Python
* Programación orientada a objetos
* Arte ASCII
* Estructuras de control y modularización

![Escape Estelar](images/escape-estelar.png)

### App de Emergencias

Aplicación diseñada para encontrar rutas óptimas hacia establecimientos de emergencia. El sistema utiliza el algoritmo Floyd-Warshall para calcular caminos mínimos dentro de una red de ubicaciones y determinar la mejor ruta posible desde el punto de origen del usuario.

Este proyecto me permitió aplicar conceptos relacionados con algoritmos, grafos y optimización de rutas para resolver un problema con aplicaciones reales.

**Tecnologías utilizadas**

* Python
* Algoritmo Floyd-Warshall
* Grafos
* Estructuras de datos

![App de Emergencias](images/app-emergencias.png)

## Tecnologías

Durante el curso de Desarrollo de Aplicaciones Web y Móviles he aprendido distintas tecnologías utilizadas para el desarrollo de aplicaciones modernas. Estas herramientas me han permitido comprender cómo construir interfaces web, agregar interactividad y organizar proyectos de manera más eficiente.

### HTML

HTML es el lenguaje utilizado para estructurar el contenido de una página web. Lo utilicé para construir mi currículum en línea publicado mediante GitHub Pages, organizando secciones como información personal, proyectos, habilidades y contacto.

### CSS

CSS permite definir la apariencia visual de una página web. Gracias a esta tecnología aprendí a personalizar colores, tipografías, distribución de elementos y diseño adaptable para diferentes tamaños de pantalla.

### JavaScript

JavaScript permite agregar funcionalidad e interactividad a las aplicaciones web. Durante el curso aprendí a manipular elementos del DOM, procesar información y responder a eventos generados por el usuario.

### Tailwind CSS

Tailwind CSS es un framework que facilita la construcción de interfaces modernas mediante clases utilitarias. Su uso permite desarrollar diseños consistentes y mantener un código más organizado durante la creación de aplicaciones web.

### Python

Python ha sido el lenguaje principal utilizado en mis proyectos personales y académicos. Fue utilizado en el desarrollo de Escape Estelar y en la App de Emergencias, permitiéndome implementar lógica de programación, algoritmos y estructuras de datos para resolver distintos problemas.

### Java

Java me ha permitido fortalecer conocimientos de programación orientada a objetos, diseño de clases y desarrollo de aplicaciones estructuradas. Su aprendizaje ha contribuido a mejorar mi comprensión sobre la organización y mantenimiento del software.

### Git y GitHub

Git y GitHub son herramientas fundamentales para el control de versiones y la gestión de proyectos. Las utilizo para almacenar mis proyectos, registrar cambios realizados durante el desarrollo y publicar contenido como mi currículum web mediante GitHub Pages.

### Aplicación de tecnologías en los proyectos

#### Escape Estelar

En este proyecto se utilizó Python para implementar toda la lógica del juego, gestionar la navegación entre habitaciones, controlar el inventario del jugador y representar los escenarios mediante gráficos ASCII dentro de la consola.

#### App de Emergencias

En este proyecto se utilizó Python para implementar el algoritmo Floyd-Warshall y procesar la información necesaria para encontrar rutas óptimas entre diferentes ubicaciones dentro de una red de caminos.

#### Currículum Web

Mi currículum web fue desarrollado utilizando HTML para la estructura del contenido y CSS para el diseño visual. Posteriormente fue publicado utilizando GitHub Pages para que pueda ser consultado desde cualquier navegador.
